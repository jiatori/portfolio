Portfolio UI/UX Process

1. Brainstorming & Grouping

  • 브레인스토밍을 통해 포트폴리오를 읽는 유저의 입장이 되어
    알고싶은 정보가 무엇일지 나열해 보았습니다.

  • Affinity Diagram 방식을 통해 나열된 데이터를
    그룹화하고 중복된 데이터는 삭제 합니다.

  • 마인드맵 작성에 참고가 되는
    중요도 높은 키워드를 추려내었습니다.

2. Mind Map

  • 네 가지 키워드를 선정하여 정보구조와 컨셉의
    바탕이 되는 마인드 맵을 작성하였습니다.

  • 작성한 마인드맵을 소개하고
    개선을 위한 아이디어를 공유하였습니다.

NAME

  • 온전히 나를 보여주는 공간 Stella Space
  • Stella 제 영어이름을 그대로 가져왔습니다.

LAYOUT & EFFECT

  • 사각형 틀에 맞춘 일관된 레이아웃이 아닌,
    우주에 두둥실 떠있는 행성의 모습처럼
    각 콘텐츠를 자유롭게 배치하였습니다.

  • 마우스와 스크롤에 따라 곡선으로 움직이는 동적효과를 적용하여
    무중력 상태로 떠있는 느낌을 표현하였습니다.

9. Coding & Test

레이아웃의 뼈대를 HTML과 CSS로 작성한 후
JS로 동적인 효과를 주었습니다.

이후 사용성 테스트와 수정을 반복하여 완성하였습니다.

  • INDEX page

    visual의 z축 스크롤 애니메이션이 특징적이며,
    slick.slide와 swiper-parallax를 활용하였습니다.

    마우스의 움직임에 따라 요소가 부드럽게 움직이며,
    이미지와 버튼에 hover 효과를 적용 하였습니다.

  • ABOUT page

    스크롤 이벤트와 라인 svg를 활용했습니다.

    Timeline은 수평 스크롤로 구현하였으며,
    Skill은 slick.slide를 활용해 흥미롭게 구현하였습니다.

  • WORK page

    visual에 counter를 활용해 직관적으로 표현하였으며
    스크롤 애니메이션을 활용해 무중력 느낌을 강조했습니다.

    NCS는 과제물의 성격에 맞게 표로 구성했습니다.

  • WORK Sub page

    결과물마다 Font,Color,Footer를 모두 다르게 적용하였으며
    바로가기 버튼, GIF, Fullpage캡쳐를 활용해 결과물을 보여주었습니다.

  • MENU

    메뉴버튼과 배경이미지로 우주 컨셉을 강조했으며,
    마우스 움직임 효과와 hover효과를 활용했습니다.
